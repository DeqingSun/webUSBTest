Follow instructions on this page to connect the counter, update population data, and update time.

Step 0. Plug the Counter

Plug the counter to a computer with a MICRO USB cable.

Step 1. Connect the Counter

Use Chrome (no other browser) to open this page.
Click "Connect" button below. It will pop up a new window, choose "Trinket M0" and click "Connect".

Function 1. Update Population Data

By default, the data below is from UN population data.
If you want to update data, get newest data from UN (https://esa.un.org/unpd/wpp/, follow steps on the PDF Manual), and copy paste data into textboxes below.
Click "Send Data" when finish editing. All population data will be sent in 5~10 seconds. Wait to see the number* changes on the counter screen.

Function 2. Update Counter's Date Time

Click "updateTime" to sync clock inside counter with your computer's time.Wait to see the date time changes on the counter screen. It will go back to population data after 10 sec.



*The number displayed on the screen is calculated by current date time and population data of this year and next year. For example, if set 2018 putuplation to 1000, 2019 population to 2000. If current date time is 2017-06-30-24:00 (in the middle of the year), the population number displayed will be 1500 ( (2000-1000)/2 +1000 = 1500 ). If currentdate date time is 2017-09-30-24:00 (3/4 of the year), the number will be 1750 ( (2000-1000)/4*3+1000 = 1750 ).
